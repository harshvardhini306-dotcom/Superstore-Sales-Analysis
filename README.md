# Superstore-Sales-Analysis
Excel analysis of retail sales data using Pivot Tables and Charts
Superstore Sale analysis using Microsoft Excel. Analysed dataset of 9,994 retail transation to identify regional performance, category profitability and seasonal sales trends. Key finding included that the west region generate 31.6% of total revenue, Technology was the most profitable category and Furniture despite being second highest in sales generated only 2.5% profit margin (profit/sales*100). 
Tools used - Pivot tables, Pivot charts in Microsoft excel
